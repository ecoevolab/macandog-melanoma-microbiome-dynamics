# Macandog melanoma microbiome dynamics

This repository contains notes, command logs, scripts, and exploratory analyses for an ecological project based on the longitudinal melanoma immunotherapy study by Macandog et al.

The original study analyzed gut microbiota dynamics in patients with melanoma undergoing anti-PD-1 immunotherapy. Patients were followed from baseline and during treatment, with clinical response categories including complete response (CR), partial response (PR), stable disease (SD), and progressive disease (PD).

## Project objective

The goal of this project is to explore the Macandog et al. dataset from an ecological perspective, focusing on microbial community dynamics during immunotherapy.

This includes:

- Understanding the structure of the available ENA data.
- Reviewing BAM files from ENA Project PRJEB61942.
- Documenting whether files correspond to 16S/amplicon or shotgun/WGS data.
- Exploring longitudinal microbiome changes across treatment time points.
- Comparing microbial patterns among response categories such as CR, PR, SD, and PD.
- Evaluating ecological concepts such as stability, persistence, turnover, and community structure.

## Current status

This repository is currently in the exploratory stage.

The first step is a technical inspection of selected BAM files to determine what information they contain and whether they are useful for downstream ecological or microbiome analyses.
